# 單字卡 WordCards

一個使用 **C# Windows Forms** 開發的英文單字學習小工具，支援單字發音、自動連續播放與背單字模式，協助記憶英文單字。



## 介紹

程式啟動時會自動讀取單字檔，左側顯示單字清單，右側顯示目前單字的**英文、音標、中文解釋**，並可播放單字發音。

## 功能說明

### 瀏覽與播放

| 操作 | 功能 |
| --- | --- |
| `Enter` | 直接選擇並播放**下一個**單字 |
| `Space` | **重複播放**目前單字的發音 |
| 點擊右側 音量 播放鍵 | 重複播放目前單字（不換字） |
| **Play** 按鈕 | 連續自動播放所有單字（再按一次變 Stop 停止） |
| 點選左側清單 | 切換到指定單字並播放 |

### 背單字模式

用來測驗自己有沒有記住單字，操作流程如下：

1. 先按 **隱藏英文** 或 **隱藏中文**，選擇要遮住的內容。
2. 再按 **背單字模式**（未先選擇遮擋會跳出提示，無法進入）。
3. 進入後，被選擇的英文／中文會被遮罩擋住。    
*選擇英文遮罩，list也會被擋住，可以使用tab切換到「隱藏中文按鈕」的下一個使用"Enter鍵"切換單字*
4. **點一下被遮住的位置 → 遮罩消失（露出答案）。**
5. **再點一次 → 遮罩重新蓋上。**
6. 再按一次 **背單字模式** 可結束此模式。

## 單字檔格式

程式預設讀取同目錄下的 `WordCards.txt`，使用 **UTF-8** 編碼，每行一個單字，欄位以 **Tab（`\t`）** 分隔：

```
單字<Tab>音標<Tab>音效檔路徑<Tab>中文解釋
```

範例：

```
abacus	ˋæbəkəs	sounds/abacus.mp3	一種利用小圓石幫助計算的工具
```

- 第 4 欄之後的內容會一併視為解釋，並以換行顯示（可放多行說明）。
- 若 `WordCards.txt` 不存在，程式會顯示錯誤訊息並結束。
- 若對應的音效檔不存在，狀態列會顯示「找無 xxx 音效檔」。
